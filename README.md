# 📦 Catalog Management System

## 📌 Overview

The **Catalog Management System** is a full-stack web application designed to manage products, categories, and orders efficiently.

This project demonstrates:

* Scalable backend development using **Spring Boot & Hibernate**
* RESTful API design
* Frontend integration using **React**
* Database handling with **MySQL**

💡 It is built to simulate real-world **e-commerce catalog management systems**.

---

## ✨ Key Features

### 🛠 Backend (Spring Boot)

* RESTful APIs for:

  * Product Management
  * Order Management

* Secure transaction handling

* Optimized database queries using Hibernate

* Category-based product organization

---

### 🎨 Frontend (React)

* API integration using Axios
* Product search functionality
* Category-based filtering
* Responsive and user-friendly UI
